# Deshi Deals E-commerce Website - GitHub Pages

This is the static version of Deshi Deals optimized for GitHub Pages deployment.

## 🚀 Live Site
Visit the live site at: https://farihasanji.github.io/Deshi-Deals/

## 📱 Features
- **Static Deployment**: Works on GitHub Pages (no server required)
- **Local Storage**: Cart data saved in browser
- **Responsive Design**: Works on all devices
- **Modern UI**: Clean, professional design
- **Coupon System**: Use "SELL200" for 20% off

## 🛠️ Technical Details
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Styling**: Tailwind CSS, DaisyUI
- **Storage**: LocalStorage for cart persistence
- **Deployment**: GitHub Pages

## 📦 Project Structure
```
docs/
├── index.html          # Main HTML file
├── index.js            # Static JavaScript (no backend)
└── deshi-deals/        # Assets folder
    └── assets/         # Product images
```

## 🔧 How It Works
- **Products**: Static data embedded in JavaScript
- **Cart**: Uses localStorage for persistence
- **Coupons**: Client-side validation
- **No Server**: Pure static website

## 🎯 GitHub Pages Setup
1. Repository Settings → Pages
2. Source: Deploy from a branch
3. Branch: main /docs folder
4. Save and wait for deployment
